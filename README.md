# Chat-O

A simple, modern web-based messaging application that enables real-time chat with direct messaging and group conversations.

## Features

- **Direct Messaging**: One-on-one conversations with other users
- **Group Chats**: Create and manage group conversations with multiple users
- **User Profiles**: Upload and customize profile pictures
- **Online Status**: See which users are currently online
- **Real-time Updates**: Real-time chats with other users

## Setup Instructions

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/Thuwayba15/ChatApp-Frontend.git
   cd ChatApp-Frontend
   ```

2. **Open in browser**
   - Open `index.html` in your web browser

## Usage Guidelines

### Creating an Account

1. Enter a username, email, and password (minimum 6 characters)
2. Click **Register** to create your account
3. You'll be automatically logged in

### Logging In

1. Enter your email and password
2. Click **Login** to access your chats

### Direct Messaging

1. Selecet a user from the list to start the conversation
2. Type your message in the input field and press the send button

### Creating a Group Chat

1. Click the **+** button in the sidebar tabs
2. Enter a group name
3. Search and select users you want to add (minimum 2 other users)
4. Click **Create Group**
5. Start messaging with the group

### Managing Your Profile

1. Click the account button in the top right
2. **Edit Profile Picture**: Click the "Edit" button next to your avatar and select an image
3. **Logout**: Click the **Logout** button
4. Click **Confirm** to return to the chat
